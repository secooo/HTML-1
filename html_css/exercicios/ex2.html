Conceitos de Web!

Internet

Rede informática utilizada para interligar computadores a nível mundial, à qual pode aceder qualquer tipo de
utilizador , e que possibilita o acesso a toda a espécie de informação.

WWW

WWW é a sigla para World Wide Web, uma rede mundial de computadores interligados.

URL

O Uniform Resource Locator, é um termo técnico que foi
traduzido para a língua portuguesa como "localizador uniforme de recursos".

Frontend

O desenvolvimento front-end da web é o desenvolvimento da interface gráfica do usuário de um site, por meio do uso de HTML, CSS
e JavaScript, para que os usuários possam visualizar e interagir com aquele site.

Backend

O backend é a estrutura que possibilita a operação do sistema.

Fullstack

O desenvolvedor full stack é aquele profissional multitarefa que cobre várias frentes na área de TI.
